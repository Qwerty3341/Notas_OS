# tar.gz
Se usa el comando
```bash
sudo tar -zxvf archivo.tar.gz
```

```sh
sudo tar -xzvf archivo.tar.gz
```
# tar bz2
```shell
tar -jxvf archivo.tar.bz2
```
# tar .gz / .tgz
```
tar -xzf archivo.tar.gz
tar -xzf archivo.tgz
```
# .tar.bz2 / .tbz / .tbz2 (bzip2)
```
tar -xjf archivo.tar.bz2
tar -xjf archivo.tbz
tar -xjf archivo.tbz2
```
# .tar.xz / .txz (xz)
```sh
tar -xJf archivo.tar.xz
tar -xJf archivo.txz
```
# tar.lz
```sh
tar --lzip -xf archivo.tar.lz
```
# tar.lzma
```sh
tar --lzma -xf archivo.tar.lzma
```
# tar.zst
```sh
tar --zstd -xf archivo.tar.zst
```
